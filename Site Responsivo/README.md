## Como baixar o projeto

```bash
# clonar repositorio
 $ git clone https://github.com/LuskaGarcia/Roadmap/tree/main/Site%20Responsivo

# entrar no diretorio
 $ cd siteResponsivo

#instalar as dependencias
 $ yarn install

#iniciar o projeto
 $ yan start
```
